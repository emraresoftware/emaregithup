# Emare GitHup

Emare yedek git hub sistemi

## Teknolojiler

Python

## Başlangıç

```bash
# Kurulum
chmod +x setup.sh && ./setup.sh

# Başlatma
chmod +x start.sh && ./start.sh

# Toplu GitHub yükleme (normal push)
python3 github_push_all.py

# Gerekirse force push
python3 github_push_all.py --force-push
```
